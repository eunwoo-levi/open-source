# Development Journal API

A simple Flask-based REST API for managing a development journal.

## Features

- Simple and lightweight Flask application
- RESTful API endpoints
- Easy to extend and customize

## Installation

1. Clone the repository:

```bash
git clone https://github.com/yourusername/development-journal-api.git
cd development-journal-api
```

2. Install dependencies:

```bash
pip install flask
```

## Usage

Run the application:

```bash
python app.py
```

The API will be available at `http://localhost:5001`

## API Endpoints

- `GET /` - Welcome message

## Requirements

- Python 3.7+
- Flask 2.0+
